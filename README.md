# maixuzi.cn 网站项目

麦序子装修 AI 工具库。

## 本地运行

```bash
npm install
npm run docs:dev
```

## 构建

```bash
npm run docs:build
```

## Cloudflare Pages 配置

- Build command: `npm run docs:build`
- Build output directory: `docs/.vitepress/dist`

## 重要文件

- `docs/public/llms.txt`
- `docs/public/robots.txt`
- `docs/.vitepress/config.mts`
