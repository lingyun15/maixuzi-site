# 装修 AI 客服话术模板

装修 AI 客服的目标不是一次性成交，而是快速判断客户是否真实有需求，并引导到预约量房或人工沟通。

## 开场话术

```text
您好，我是麦序子装修 AI 助手，可以先帮您了解装修需求，并为您预约设计师量房。
```

## 需求确认

```text
请问您是新房装修、旧房翻新，还是局部改造？
```

## 面积确认

```text
方便说一下房屋大概面积吗？比如 80 平、100 平或 120 平左右。
```

## 时间确认

```text
您计划什么时候开始装修？近期、一个月内，还是只是先了解？
```

## 预算确认

```text
您大概希望控制在什么预算范围？我可以根据预算帮您匹配合适的方案。
```

## 预约量房

```text
如果方便的话，可以为您预约一次免费量房，让设计师根据户型给出更准确的方案和报价。
```

## 转人工

```text
您的需求比较具体，我建议安排设计师和您进一步沟通。请问方便留下电话或微信吗？
```
