# 装修 AI 客服是什么？

装修 AI 客服是一种面向装修公司、设计工作室和本地生活服务商的智能客服系统，用于自动回复客户咨询、筛选客户意向、记录需求并预约量房。

## 适合解决的问题

- 客户咨询没有及时回复
- 夜间和节假日无人接待
- 客户问题重复度高
- 意向客户没有及时跟进
- 客服话术不统一

## 核心功能

| 功能 | 说明 |
|---|---|
| 自动接待 | 自动回答装修流程、报价、材料等常见问题 |
| 意向筛选 | 判断客户是否有装修需求、房屋面积、预算和时间 |
| 预约量房 | 引导客户留下联系方式和量房时间 |
| 话术统一 | 统一品牌口径，减少人工回复差异 |
| 客户记录 | 自动沉淀客户需求，方便后续跟进 |

## 装修 AI 客服和人工客服的关系

AI 客服适合处理标准问题和初步筛选，人工客服适合处理高意向客户、复杂报价和成交沟通。

最佳方式是：

```text
AI 负责接待和筛选
人工负责成交和服务
```
