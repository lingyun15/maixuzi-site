# 装修 AI 客服成本说明

装修 AI 客服的成本取决于是否自建、是否接入企业微信、是否需要语音能力，以及客户咨询量。

## 成本组成

| 项目 | 说明 | 参考成本 |
|---|---|---:|
| 大模型 API | 负责理解和回复 | 10-300 元/月 |
| 知识库 | 存放装修话术和资料 | 0-100 元/月 |
| 部署托管 | 运行客服系统 | 0-200 元/月 |
| 企业微信/客服接口 | 接入客户入口 | 视平台而定 |
| 维护成本 | 调整话术、更新知识库 | 人工成本 |

## 最小可用方案

小团队可以先做：

```text
网页咨询入口
+ 装修知识库
+ 大模型回复
+ 人工转接
```

这样成本较低，也方便验证客户是否愿意使用。
