# 装修工天管理工具功能设计

第一版工天管理工具不追求复杂，而是优先解决每天记工天、月底算工资和项目人工成本统计的问题。

## v1.0 核心功能

1. 项目管理
2. 工人管理
3. 工种管理
4. 每日工天记录
5. 自动计算工资
6. 按项目统计
7. 按工人统计
8. 导出 Excel

## 首页仪表盘

展示：

- 今日工天数
- 本月总工天
- 本月人工成本
- 未结算工资
- 进行中项目数
- 工人数量

## 工天记录

字段包括：

- 日期
- 项目
- 工人
- 工种
- 工天数量
- 单价
- 金额
- 是否结算
- 备注

金额自动计算：

```text
金额 = 工天 × 单价
```

## 导出 Excel

支持导出：

- 项目工天明细表
- 工人工资结算表
- 月份人工成本表
- 未结算工资表
