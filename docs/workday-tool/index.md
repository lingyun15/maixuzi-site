# 装修工地工天管理工具

装修工地工天管理工具用于记录每天每个工人在不同项目上的工天、单价和工资，并自动统计项目人工成本和工人工资。

## 适合谁使用？

- 装修公司老板
- 工长
- 项目经理
- 施工队
- 设计工作室

## 解决什么问题？

- 防止漏记工天
- 自动计算工资
- 快速统计项目人工成本
- 月底一键导出工资表
- 减少人工算账错误

## 功能清单

| 功能 | 说明 |
|---|---|
| 项目管理 | 管理多个装修工地 |
| 工人管理 | 记录工人、工种和单价 |
| 工天记录 | 每天记录工天数量 |
| 工资统计 | 自动计算工资 |
| Excel 导出 | 导出结算表 |

## 常见问题

### 工天可以记 0.5 天吗？

可以，支持 0.5、1、1.5、2 和自定义工天。

### 可以按项目统计人工成本吗？

可以，每个项目都可以查看总工天、总工资、已结算和未结算金额。

### 需要安装软件吗？

网页版不需要安装，打开网页即可使用。
