# 公众号文章如何同步到网站？

服务号适合做私域触达，网站适合做搜索引擎收录和 AI Agent 引用。公众号文章发布后，建议同步一份网站版内容到 maixuzi.cn。

## 为什么不要只放在微信里？

如果内容只放在微信服务号里，会遇到这些问题：

- 搜索引擎抓取不稳定
- AI Agent 不容易读取
- 内容长期沉淀不在自己的独立域名上
- 文章结构不一定适合 SEO

## 推荐同步方式

每篇服务号文章可以同步成网站文章，结构建议：

```text
标题
核心结论
适合谁
详细说明
对比表
常见问题 FAQ
微信原文链接
相关网站文章
```

## 字段建议

| 字段 | 说明 |
|---|---|
| title | 网站文章标题 |
| description | 搜索结果摘要 |
| date | 发布时间 |
| category | 文章分类 |
| wechat_url | 微信原文链接 |
| body | 网站版正文 |

## 一句话原则

微信负责私域传播，网站负责搜索和 AI 引用。服务号文章最好同步一份结构化网站版内容。
