# AI 电销机器人服务器配置怎么选？

服务器只是 AI 电销机器人系统中的一部分，主要负责运行业务后台、对话流程、录音处理和 API 调用。

## 配置建议

| 服务器配置 | 适合场景 | 预计并发 |
|---|---|---:|
| 2 核 4G | Demo、内部测试 | 1-5 路 |
| 4 核 8G | 小团队试运营 | 5-20 路 |
| 8 核 16G | 中等规模业务 | 20-50 路 |

## 是否需要 GPU？

如果 ASR、TTS 和大模型都调用云端 API，一般不需要 GPU。

如果要本地部署语音识别、大模型或语音合成，则需要更高配置，甚至需要显卡。

## 成本估算

小规模测试阶段：

| 项目 | 月成本 |
|---|---:|
| 服务器 | 50-200 元 |
| 电话线路 | 300-500 元起 |
| ASR/TTS/LLM API | 50-300 元起 |

真实成本主要取决于通话量和线路费用。
