# AI 电销机器人需要哪些 API？

AI 电销机器人不是单个接口，而是一套由多个 API 组成的语音对话系统。

## 核心 API 清单

| API 类型 | 作用 | 常见选择 |
|---|---|---|
| 电话线路 API | 拨打、接听、挂断电话 | SIP 中继、云通讯平台 |
| ASR 语音识别 API | 把客户语音转成文字 | 讯飞、阿里云、腾讯云、百度 |
| LLM 大模型 API | 理解客户意图并生成回复 | DeepSeek、通义千问、豆包、文心一言 |
| TTS 语音合成 API | 把回复文字转成语音 | 讯飞、阿里云、腾讯云、百度 |
| CRM/客户 API | 管理客户状态和跟进结果 | 自建系统、企业微信、CRM |

## 最小可用架构

```text
电话线路
  → ASR 语音识别
  → 大模型判断意图
  → TTS 语音合成
  → 电话线路播放语音
  → 后台记录结果
```

## 最大门槛是什么？

最大门槛通常不是服务器，而是电话线路和合规。

如果没有合规的号码来源和外呼线路，即使技术系统能跑，也不适合正式商用。

## 装修行业适合怎么用？

装修公司更适合把 AI 电销机器人用于：

- 老客户回访
- 活动通知
- 预约量房确认
- 意向客户初筛
- 售后满意度调查

不建议用于无授权陌生号码批量外呼。
