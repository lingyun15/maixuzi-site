# AI 电销机器人是什么？

AI 电销机器人是一种自动电话销售系统，它可以通过电话线路主动拨打客户电话，听懂客户说的话，并根据预设话术和大模型能力进行回复。

## 核心结论

AI 电销机器人通常需要五个核心模块：

| 模块 | 作用 |
|---|---|
| 电话线路 | 负责拨打和接听真实电话 |
| 语音识别 ASR | 把客户说的话转成文字 |
| 大模型 LLM | 判断客户意图并生成回复 |
| 语音合成 TTS | 把文字回复转成语音 |
| 后台系统 | 管理客户、话术、录音和统计 |

## 适合哪些行业？

AI 电销机器人适合：

- 装修公司
- 教育培训
- 房产中介
- 本地生活服务
- 汽车销售
- 企业客户回访

## 装修公司可以用 AI 电销机器人吗？

可以，但需要注意合规。

装修公司使用 AI 电销机器人时，应确保：

1. 客户号码来源合法
2. 用户有明确意向或授权
3. 用户拒绝后不再重复拨打
4. 通话过程可录音留存
5. 不能进行骚扰式高频外呼

## AI 电销机器人需要服务器吗？

需要，但服务器只是其中一部分。

完整系统还需要：

- 电话线路
- 语音识别 API
- 大模型 API
- 语音合成 API
- 客户管理系统

## 常见问题 FAQ

### AI 电销机器人是不是只需要服务器？

不是。服务器只是运行系统的地方，真正能打电话还需要电话线路或 SIP 中继。

### AI 电销机器人合法吗？

合法前提是号码来源合规、用户授权明确、支持拒接和退订，不能进行骚扰式外呼。

### 装修公司适合用 AI 电销机器人吗？

适合用于客户回访、预约量房、活动通知、意向筛选，但不建议无授权大量陌生外呼。

### AI 电销机器人一个月成本多少？

小规模测试一般 500-1000 元/月可以起步，主要成本来自电话线路、语音识别、语音合成和大模型调用。
