# SketchUp 插件推荐清单

SketchUp 插件不需要装很多，先把常用插件学透更重要。

## 推荐插件

| 插件 | 用途 | 优先级 |
|---|---|---|
| LibFredo6 | Fredo6 插件公共库 | 高 |
| JointPushPull | 联合推拉、复杂面推拉 | 高 |
| CleanUp³ | 清理废线、优化模型 | 高 |
| 坯子库 / PiziTools | 国内常用插件合集 | 高 |
| 1001bit Tools | 建筑构件快速生成 | 中 |
| Profile Builder | 踢脚线、线条、构件生成 | 中 |
| Curviloft | 曲面放样 | 中 |
| RoundCorner | 一键圆角 | 中 |

## 插件学习原则

1. 不要一次装太多插件
2. 优先学建模思路，再学插件
3. 常用插件要读官方文档
4. 插件只是加速器，不是建模能力本身
