# D5 全景图怎么做？

D5 渲染器支持直接输出全景图和 VR 漫游，非常适合装修公司给客户展示方案。

## 基本流程

```text
1. SketchUp 完成建模
2. 导入 D5 Render
3. 调整材质和灯光
4. 设置全景相机点位
5. 输出 8192×4096 全景图
6. 生成 VR 漫游链接
```

## 点位建议

全屋全景一般选择：

- 客厅中心
- 餐厅中心
- 主卧中心
- 次卧中心
- 厨房门口或中心
- 卫生间门口或中心

## 常见问题

### 全景图分辨率用多少？

建议至少使用 8192×4096。

### 客户怎么看？

可以生成 VR 链接或二维码，通过微信发给客户查看。
