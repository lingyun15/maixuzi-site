# SketchUp + D5 学习路线

SketchUp + D5 是当前室内设计师非常适合学习的一套工作流。SketchUp 负责自由建模，D5 负责实时渲染、材质灯光和全景漫游。

## 推荐学习顺序

```text
1. SketchUp 基础操作
2. 群组与组件
3. CAD 导入与户型建模
4. 吊顶、门窗、柜体建模
5. 材质贴图
6. D5 导入模型
7. D5 材质、灯光、相机
8. 效果图与全景图输出
```

## 为什么酷家乐设计师适合升级到 SketchUp + D5？

| 对比项 | 酷家乐 | SketchUp + D5 |
|---|---|---|
| 建模自由度 | 中等 | 高 |
| 出图质感 | 中等 | 高 |
| 全景漫游 | 有 | 强 |
| 可控性 | 中等 | 高 |
| 学习成本 | 低 | 中等 |

## 核心建议

先用一套真实户型完整跑通：

```text
CAD → SketchUp 建模 → D5 渲染 → 全景漫游 → 作品集
```
